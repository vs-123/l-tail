* L-TAIL

*L-Tail* stands for *Log-Tail*. L-Tail is a simple, memory-efficient, POSIX-compliant, high-performance, log-scanning utility tool for stream monitoring. This project utilises buffered I/O and non-blocking polling to provide real-time stream monitoring with a minimal resident set size.
